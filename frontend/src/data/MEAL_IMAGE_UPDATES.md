# Meal Image Updates Guide

This document tracks recipe site images found for meals. Pattern to follow:
- Search Google: "[meal name] recipe site:tipbuzz.com OR site:bellyfull.net OR site:thefoodcharlatan.com"
- Find the recipe page
- Locate the main recipe image URL
- Update mealImages.ts with the direct image URL
